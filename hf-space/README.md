# LLM From Scratch

Interactive demos of tiny language models trained in the [LLM From Scratch](https://nipunbatra.github.io/llm-from-scratch/) educational series.

## Features

- **Name Generator**: Character-level model trained on Indian names
- **Shakespeare**: Transformer model generating Bard-style text

## Model Scale

These are intentionally tiny models (~600K parameters) for educational purposes:

```
These models:   ~600K parameters
GPT-2 Small:    124M parameters (200x larger)
GPT-4/Claude:   ~1T+ parameters (1,600,000x larger)
```
